# 鬼藏素材配图 Skill 使用指南

guizang-material-illustration 是一个给文章配图的 Skill，用于为技术文档、博客文章、工作汇报等内容自动生成或推荐合适的插图素材。

## 前置检查

确认 Skill 已安装并可用，调用前检查可用工具列表中是否有 `guizang-material-illustration` 相关能力。

## 安装配置

该 Skill 来自 GitHub 仓库：

```
https://github.com/op7418/guizang-material-illustration
```

在 TRAE/CodeBuddy 的 Skill 配置中添加：

```yaml
- name: guizang-material-illustration
  description: 给文章配图的 Skill，用于为文章自动生成或推荐插图素材
  repo: https://github.com/op7418/guizang-material-illustration
```

## 使用说明

### 使用场景

| 场景 | 说明 |
|------|------|
| 技术文档配图 | 为 API 文档、技术方案、架构说明等添加示意图 |
| 博客文章配图 | 为技术博客、行业分析文章添加封面图和插图 |
| 工作汇报配图 | 为周报、月报、项目总结添加图表和视觉元素 |
| 演示文稿配图 | 为 PPT、幻灯片添加背景图和装饰元素 |

### 使用步骤

1. **理解文章内容**：阅读需要配图的文章，识别关键主题和重点内容
2. **确定配图需求**：分析需要什么样的图片（封面图、流程图、示意图、装饰图等）
3. **调用配图 Skill**：使用 `guizang-material-illustration` 根据文章内容生成或推荐配图
4. **选择合适图片**：从推荐结果中选择最符合文章主题的图片
5. **插入到文章**：将选中的图片插入到文章的合适位置

### 配图策略

| 类型 | 适用场景 | 示例 |
|------|---------|------|
| 封面图 | 文章开头 | 与主题相关的视觉化标题图 |
| 流程图 | 流程说明 | 系统架构图、业务流程图 |
| 示意图 | 概念解释 | 抽象概念的可视化表达 |
| 数据图 | 数据展示 | 图表、统计图形 |
| 装饰图 | 段落分隔 | 与内容相关的小图标或分割线 |

## 完成标准

- [ ] 已阅读文章内容并理解主题
- [ ] 已识别需要配图的位置和类型
- [ ] 已调用 `guizang-material-illustration` Skill 生成配图
- [ ] 已选择合适的图片并插入到文章中
- [ ] 配图与文章内容相关且提升可读性
